Everyday Ingredients

All of our sample recipes are built around pantry staples like pasta, rice, eggs, and vegetables. When you add your own recipes later, you can keep the same idea: simple food that doesn’t require a special trip to the store.

Beginner-Friendly

The layout of this site is intentionally clear and minimal. Every card highlights prep time, difficulty, and a short description so visitors can quickly spot what works for them.

Made as a Class Project

CookBook was created as a student project using only HTML and CSS. It shows how much you can do with basic web technologies when you combine clean design with a simple idea.
